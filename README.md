# Flixplorer

Flixplorer is a global search engine for Netflix content. Users can create an account and log in to add titles to their favorites list. Upon startup, the program automatically loads the latest TV shows and movies from Netflix. Users can also search for content directly using the search field. Additionally, users can watch their chosen title directly on Netflix by clicking on the "Watch on Netflix" button within the program.

# Using the Program

This repository includes an executable file that you can run to use the program without having to build it yourself. I understand that some users may still want to review the source code before running the program. If you have any doubts or concerns about the executable, please refer to the source code which is available in this repository.

In this case, I have included the executable because the source code includes sensitive information such as API keys that cannot be added to GitHub. To ensure that the program works as intended and to protect the sensitive information, I have compiled the code and provided the executable.
